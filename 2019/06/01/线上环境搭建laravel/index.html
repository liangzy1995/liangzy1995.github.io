线上环境搭建laravel

服务器要求

  • PHP >= 7.1.3
  • OpenSSL PHP 拓展
  • PDO PHP 拓展
  • Mbstring PHP 拓展
  • Tokenizer PHP 拓展
  • XML PHP 拓展
  • Ctype PHP 拓展
  • JSON PHP 拓展
  • BCMath PHP 拓展

[本地环境Homestead，过于庞大，每次启动vagrant繁琐，后换成Valet。]:

安装BCMath

PHP7以上版本不在默认安装BCMath拓展

编译安装
        1.进入PHP源码包目录下的ext/bcmath目录。
        2.执行phpize命令，phpize命令在PHP安装目录的bin目录下，/usr/bin/phpize。
        3.yum install php72w-devel #安装php-devel
        3.执行./configure --with-php-config=/usr/bin/php-config。
        4.执行make && make install。
        6.在Dynamic Extensions配置块下添加一行extension=bcmath.so。
        7.systemctl restart php-fpm 重启php-fpm

安装composer

curl -sS https://getcomposer.org/installer | php
mv composer.phar /usr/local/bin/composer
# 设置中国镜像
composer config -g repo.packagist composer https://packagist.phpcomposer.com

安装Git工具

yum install -y git
git --version
git config --global user.name "aliyuncentos"
git config --global user.email "aliyuncentos@gmail.com"
# 生成SSH Key
ssh-keygen -t rsa -C "aliyuncentos@gmail.com"
